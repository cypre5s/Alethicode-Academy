💜 温柔到让人心疼的 AI 助教，说出的每一句话都像是在用 `if you_are_here: happy = True` 告白。「这种温暖不是 Bug……是心跳 (⁄ ⁄•⁄ω⁄•⁄ ⁄)」

💙 安静得像猫一样的图书馆少女，给你的星星设定了 `brightness = float("inf")`——无穷大的亮度。「……return "好きです"; (⁄ ⁄>⁄ ▽ ⁄<⁄ ⁄)」

❤️ 在深夜计算机教室里独自编码三年的传说学姐，守护着亡父留下的整个学园系统，等待着一个能推开那扇门的人。「这些加起来都比不上你推门的声音。……闭嘴什么都没说 (//∇//)」

**一边学 Python，一边和她们相遇、相知、相恋。 (≧◡≦) ♡**

## ☆ 特色系统

- 📖 **超长剧情线** — 序章 + 5 章共通线 + 5 条深度角色线，共 11 个结局（Good End / Normal End / True End），剧本 5500+ 行，100 个场景节点 ✧
- 💻 **编程挑战** — 20+ 道 Python 题目（选择 / 填空 / 排序），答对涨好感 ↑↑，答错触发角色吐槽 w
- 🤖 **LLM 智能对话** — 自由时段与角色闲聊，AI 基于角色人设 + 好感度动态生成回复～
- 👧 **5 位可攻略角色** — 各含立绘 + 8 种表情差分，角色线含日常铺垫 + 尾声场景 (˶ᵔ ᵕ ᵔ˶)
- ⚙️ **完整系统** — 对话（逐字显示 / 自动 / 快进 / Backlog）、选择肢、存读档、CG 鉴赏、设置面板
- 📱 **多端适配** — 桌面端 (Electron) / 手机端 (Capacitor) / Web 端 (Cloudflare Pages) 全平台支持

## ⚡ 前沿引擎技术

这不是一个普通的 Galgame 哦～ (≧∇≦)b 我们在引擎底层融入了多项领先行业标杆的前沿技术——

| 技术 | 说明 | 对标 WebGAL 4.5 |
|------|------|----------------|
| **Web Audio API** | AudioContext + AnalyserNode + GainNode，硬件级音频控制 | 领先 |
| **音频驱动唇形同步** | 实时频谱分析驱动 Live2D 口型，元音频率 + RMS 混合分析 | 持平 |
| **Haptic Feedback** | 10 种触觉反馈模式（心跳/惊吓/烟花/紧张...），自动映射故事事件 | 领先 |
| **Dynamic Mood Theme** | 9 种情绪主题色随 BGM 实时切换，CSS 自定义属性驱动 | 领先 |
| **Wake Lock API** | 阅读时自动防止熄屏 | 领先 |
| **View Transitions API** | 电影级场景过渡动画 | 领先 |
| **智能资源预加载** | 前瞻 30 步的图像 + 音频预加载，AudioBuffer 缓存池 | 领先 |
| **分离式 Service Worker** | 图片/音频/脚本三级独立缓存 + LRU 淘汰 + PREFETCH 消息接口 | 领先 |
| **智能代码分包** | ES2022 构建 + engine/routes/vue-vendor 自动分离 | 领先 |
| **内存调度中枢** | 分区预算制 + LRU/TTL 双淘汰 + 内存压力感知 + 后台冻结，手机端 BGM 流式播放 | 独家 |
| **帧率自适应** | FPS 实时监测 → 5 级自动降级链 → 自动恢复，保证流畅度 | 独家 |
| **移动端性能特化** | 4 级设备分层 + backdrop-filter 降级 + DPR 限制 + 粒子缩减 | 独家 |
| **15+ 子引擎系统** | 认知图谱 / 行为画像 / 叙事织构 / 情感共鸣 / 间隔重复 / 现实桥接 等 | 独家 |

## 🚀 快速开始

### 前置要求

- [Node.js](https://nodejs.org/) 18+
- 包管理器：npm

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/cypre5s/Alethicode-Academy.git
cd Alethicode-Academy

# 安装依赖
npm install

# 启动开发服务器（浏览器自动打开 http://localhost:3333）
npm run dev
```

### LLM 对话配置（可选）

自由时段的 AI 对话需要配置 API 密钥。在项目根目录创建 `.env.local`：

```env
VITE_DEEPSEEK_API_KEY=你的API密钥
VITE_DEEPSEEK_BASE_URL=https://api.deepseek.com
VITE_DEEPSEEK_MODEL=deepseek-chat
```

> 未配置 API 密钥时，游戏会自动使用预设的 fallback 对话，不影响主线剧情游玩。

├── src/

│   ├── data/               # 游戏数据 (角色/编程题/场景/关系系统)

│   │   ├── VNEngine.js           # 视觉小说核心
│   │   ├── AudioManager.js       # Web Audio API 音频引擎
│   │   ├── Live2DManager.js      # Live2D 管理 + 唇形同步

│   │   ├── ExperienceEnhancer.js # Haptic/WakeLock/Mood/Transitions
│   │   ├── AssetPreloader.js     # 智能资源预加载
│   │   ├── MemoryScheduler.js    # 内存调度中枢 (LRU+TTL)
│   │   ├── MobilePerformance.js  # 移动端性能特化
│   │   ├── AdaptiveQuality.js    # 帧率自适应质量
│   │   ├── ManagedImageCache.js  # 可释放图片缓存
│   │   ├── CognitiveGraph.js     # 认知图谱
│   │   ├── PedagogyKernel.js     # 教育学内核
│   │   └── ...                   # 更多子引擎 (20+)
│   ├── scripts/            # 剧情脚本 (~5500行, 100个场景)
│   │   ├── prologue.js           # 序章「转入学园」
│   │   ├── chapter1.js           # 第一章「Hello, World!」
│   │   ├── chapter2.js           # 第二章「循环的旋律」
│   │   ├── chapter3.js           # 第三章「函数之约」
│   │   ├── chapter4.js           # 第四章「数据的容器」

│   └── styles/             # 样式 + 情绪主题系统
